PhData: All of the Heatmaps

Hi! Welcome to the site I initially made for ease of analysis of my research data. I didn't intend for this to be used or viewed by anyone else, so I apologise for the mess. Each page on this site contains the filterable heatmaps for my PhD research. Below is a bit of a guide so you can find your way around.

  • ind21: The heatmaps for the Interactive Narrative Design 2021 Discord Server public channels.
  • ind23: The heatmaps for the Interactive Narrative Design 2023 Discord Server public channels.
  • ind21: The heatmaps for the Brisbane Global Game Jam 2023 Server public channels.
  • ind21: A side-by-side comparison of the IND21 and IND23 public channels. Note: this page works best on Desktop

This data is from my PhD, recorded 2021-2023. Copyright Jess Greentree, 2025.
